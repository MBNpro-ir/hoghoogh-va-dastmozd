# 📝 گزارش تغییرات HvM

## 🚀 نسخه 0.4.0 alpha — ۱۴۰۵/۰۴/۰۲

### 📦 عملیات دسته‌ای
- بخش ورود دسته‌ای کارکنان به دو حالت «نمایش و ویرایش اطلاعات قبلی» و «ثبت اطلاعات جدید» تقسیم شد.
- در حالت نمایش اطلاعات قبلی، امکان ویرایش، حذف، کپی‌کردن و افزودن ردیف‌های جدید کنار داده‌های موجود فراهم شد.
- مقادیر خودکار، وابستگی عددها و تیک‌های قابل روشن/خاموش مانند فرم تک‌کارمند در جدول دسته‌ای اعمال می‌شوند.
- ستون‌های تاریخ با اعداد فارسی نمایش داده می‌شوند.

### 🪟 تجربه ویندوز
- منوی سمت راست ویندوز به‌صورت نرم و انیمیشنی جمع و باز می‌شود.
- حالت جمع‌شده به نوار آیکونی تبدیل می‌شود و وضعیت آن برای اجرای بعدی ذخیره می‌ماند.
- هدر، فوتر، انتخاب تم و Tooltipهای منو برای حالت فشرده بهینه شدند.

### ♿ دسترسی‌پذیری و ظاهر
- گزینه «اندازه رابط برنامه» اضافه شد تا کل UI برنامه کوچک و بزرگ شود.
- تگ‌های «جدید» از بخش‌های تنظیمات حذف شدند.
- ظاهر تمام Sliderهای برنامه به طراحی Material 3 Expressive اندروید 16 تغییر کرد.
- تنظیمات قدیمی دسترسی‌پذیری با مقدار پیش‌فرض امن برای مقیاس رابط سازگار می‌مانند.

### 🧪 کیفیت
- تست‌های جدید برای عملیات دسته‌ای، منوی جمع‌شونده، مقیاس رابط و ظاهر جدید Slider اضافه شد.
- اجرای موفق `flutter analyze`.
- اجرای موفق `flutter test`.

### ⚙️ انتشار
- نسخه برنامه به `0.4.0 alpha` ارتقا یافت.
- انتشار از طریق GitHub Actions و تگ نسخه به‌صورت Pre-release انجام می‌شود.
